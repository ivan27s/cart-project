# CartProject

### Запуск и установка в контайнере Docker:

    docker-compose up --build

### или (если не установлен Docker):
  установка зависимостей:

    npm install
  запуск в режиме разработки:
   
    npm run start
### в браузере открываем:
    http://localhost:4200
    
### Build

    npm run build
